# CI Proof Project

This project demonstrates a simple CI pipeline using Jenkins, Node.js, and Jest.

## Project Structure
- `app/`: Contains the application logic.
- `tests/`: Contains Jest test cases (including intentional failures).
- `reports/`: Target directory for test reports.
- `Jenkinsfile`: Defines the CI pipeline stages.

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run tests:
   ```bash
   npm test
   ```
